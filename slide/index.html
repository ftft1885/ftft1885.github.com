HTML模板引擎制作

什么是HTML模板引擎？

模板引擎分后端渲染和前端渲染两种，我们熟知的php和asp属于后端渲染模板

模板的基本功能就是定义一个新的标签（或其他形状），允许在该标签中加入可执行代码

HTML模板内嵌可执行代码简单demo

<ul>
<%  for(var i = 0; i < year.length; i++){ %>
    <li><%year[i]%>年大吉</li>
<% } %>
</ul>

MVVM框架制作

什么是MVVM？

  • M ===> Model
  • V ===> View
  • VM ===> ViewModel

MVVM的特点与功能

  • 0 DOM操作
  • Model与View的数据双向绑定
  • 复杂逻辑表单处理（如嵌套）
  • 提供监听函数，可自由处理
